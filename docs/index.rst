Welcome to Cethil Acquisition's documentation!
==============================================

The Cethil Acquisition project aims to bring solutions for instrument control and data acquisition through Python language.

It will focus on Keithley instruments (NI daq not implemented yet) and will be using a python module called Pymodaq.
